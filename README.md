# R&E Produções - Convites Personalizados

Single-page e-commerce application for custom invitation design.

## Quick Start

1. Open `index.html` in any web browser
2. No build step or installation needed

## Features

- User registration and login (localStorage-based)
- Interactive product catalog
- Shopping cart management
- Order history
- Direct WhatsApp integration

## Project Structure

```
├── index.html          # Main application file (HTML + CSS + JS)
├── CLAUDE.md          # Development guide for AI assistants
└── .github/workflows/ # GitHub Actions for auto-sync
```
